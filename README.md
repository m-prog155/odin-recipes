# odin-recipes
This project is a basic recipe website built with HTML.

The website contains a homepage with links to multiple recipe pages. Each recipe page includes an image, a description, a list of ingredients, and step-by-step cooking instructions.

## Skills Demonstrated

By completing this project, I have demonstrated the ability to:

* Create and structure web pages using HTML
* Organize files and directories in a web project
* Create links between multiple pages
* Add images to web pages using relative paths
* Use Git and GitHub for version control
